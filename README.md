# Kit de Scripts para SysAdmin

Este projeto reúne scripts em Bash desenvolvidos para
automatizar tarefas comuns de administração de sistemas Linux.

## Objetivo
Criar um conjunto reutilizável de scripts para auxiliar
rotinas diárias de um SysAdmin.

## Cenário
Servidor Linux que necessita de scripts simples para
monitoramento, manutenção e verificação do sistema.

## O que foi implementado
- Script para verificação de uso de disco
- Estrutura organizada de scripts
- Boas práticas de comentários
- Execução simples via terminal

## Estrutura do repositório
configs/
  disk_usage.sh
docs/
  README.md

## Tecnologias utilizadas
- Linux
- Bash
- Git & GitHub

## Resultados
Scripts funcionais que podem ser reutilizados
em diferentes ambientes Linux.
